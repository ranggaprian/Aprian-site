HTML

HTML is the markup language of the internet (web) in the form of codes and symbols that are added to a file intended for display in a website. In short, HTML is a markup language used to create websites. Website created with HTML, can be seen by everyone connected to the Internet. Of course by using the internet browser application (browser) such as Internet Explorer, Mozilla Firefox and Google Chrome.

CSS

CSS stands for Cascading Style Sheets. Contains a series of instructions that determine how a text will appear on a web page. Designing text design can be done by defining fonts (letters), colors (colors), margins (size), background (background), font size (font sizes) and others. Elements such as colors, fonts, sizes and spacing are also called "styles".

PHP is the most widely used script programming language today. PHP is widely used to program dynamic websites, although it is not possible to use for other uses. Popular examples of PHP applications are forums (phpBB) and MediaWiki (software behind Wikipedia).
